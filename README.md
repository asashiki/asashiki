## About

I build small, personal systems that make AI agents more useful in daily life.

Current focus:

- MCP services that render real UI inside AI chats.
- Personal data infrastructure: device timeline, health/location/activity logs, and archive tools.
- Lightweight web apps, blog systems, and automation around my own workflows.
- Experiments that connect LLMs with music, stickers, voice, choice wheels, and long-term memory.
